## Zero-shot の例

```
以下のイタリアンレストランの利用者のレビューを「ポシティブ」か「ニュートラル」か「ネガティブ」に分類して下さい。

利用者のレビュー
このレストランは料金は高かったが、料理美味しかった。
```

---

## Few-shot の例

```
以下のイタリアンレストランの利用者のレビューについて例を参考にして「ポシティブ」か「ニュートラル」か「ネガティブ」に分類して下さい。

利用者のレビュー
このレストランは料金は高かったが、料理は美味しかった。

例）
利用者のレビュー
このレストランは騒がしい雰囲気だったが料理は美味しかった。
結果）ネガティブ評価とポシティブ評価の両方があるのが、「騒がしい雰囲気だったが」という文に続いて最後に「美味しかった」とポシティブな意見で締めくくっているので、「ポシティブ」と判定できる
```

## Chain-of-thought (CoT)  の例

(1) CoT を使わない場合
```
AはBより背が高い。CはBより背が低い。DはCより背が高いがBより背が低い。
この4人を背の高い順に並べてください。答えだけを記号で。
```

(2) CoT を使う場合
```
AはBより背が高い。CはBより背が低い。DはCより背が高いがBより背が低い。
この4人を背の高い順に並べてください。ステップバイステップで考えてから答えてください。
```

* 想定する回答
- (1) は A > D > B > C (誤答)
- (2) は A > B > D > C (正答)
